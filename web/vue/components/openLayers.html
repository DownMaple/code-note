OpenLayers 在 Vue3 项目中的应用 ​

基于项目实际使用场景整理，涵盖地图初始化、点位标记、图形绘制、扇形展示、聚合、交互等功能。项目使用 OpenLayers v10.7.0 + 天地图底图。

核心概念 ​

在开始使用 OpenLayers 之前，需要理解几个核心概念：

  • Map：地图容器，承载所有图层、交互和控件，需要绑定到一个 DOM 元素上。
  • View：视图，控制地图的中心点、缩放级别、旋转角度和投影方式。一个 Map 只有一个 View。
  • Layer（图层）：地图内容的载体，按类型分为瓦片图层（TileLayer，加载底图）和矢量图层（VectorLayer，承载自定义要素）。图层按 zIndex 控制叠放顺序。
  • Source（数据源）：图层的数据提供者。XYZ 提供瓦片地址，VectorSource 存放矢量要素，Cluster 在 VectorSource 基础上实现聚合。
  • Feature（要素）：地图上可交互的单个对象（一个点、一条线、一个面），由几何体（Geometry）和属性数据组成。
  • Geometry（几何体）：定义要素的空间形状，常用的有 Point（点）、LineString（折线）、Polygon（多边形）。
  • Style（样式）：控制要素的视觉表现，包括 Icon（图标）、Fill（填充）、Stroke（描边）、Text（文字标注）等。可以是静态对象，也可以是根据要素属性动态返回的函数。
  • Interaction（交互）：用户与地图的交互行为，如 Draw（绘制）、Translate（拖拽）等。通过 addInteraction / removeInteraction 动态启停。
  • Overlay（覆盖物）：将任意 DOM 元素锚定到地图坐标上，常用于弹窗、Tooltip。与图层不同的是，Overlay 直接操作 DOM。
  • Projection（投影）：OpenLayers 默认使用 EPSG:3857（Web 墨卡托）投影，而业务数据通常是 EPSG:4326（WGS84 经纬度）。两者的转换贯穿所有地图操作——数据传入地图用 fromLonLat，从地图取出用 toLonLat。

地图初始化 ​

天地图底图 ​

天地图提供四种图层服务，通常成对使用：

类型底图标注说明
影像img_wcia_w卫星影像 + 中文地名标注
矢量vec_wcva_w矢量地图 + 中文地名标注

底图提供地理信息，标注图层叠加在上方显示地名、道路名等文字。两者缺一不可。

js
function createTianDiTuLayer(type, tk) {
  return new TileLayer({
    source: new XYZ({
      url: `https://t0.tianditu.gov.cn/DataServer?T=${type}&x={x}&y={y}&l={z}&tk=${tk}`,
      crossOrigin: 'anonymous', // 允许跨域，截图功能需要
      wrapX: false              // 禁止水平重复
    })
  })
}

tk 是天地图开发者 Key，在 天地图控制台 申请。离线部署时将 URL 替换为本地瓦片服务地址即可，图层代码无需改动。

Vue3 地图组件 ​

在 Vue3 中使用 OpenLayers 有一个关键注意点：地图实例不能放在响应式数据中。Vue 的 reactive / ref 会对对象进行深度代理（Proxy），而 OpenLayers 的 Map 对象内部结构复杂，被代理后会导致严重的性能问题甚至报错。应使用 shallowRef 或普通变量存储。

vue
<template>
  <div ref="mapRef" class="map-container" />
</template>

<script setup>
import { ref, onMounted, onBeforeUnmount } from 'vue'
import Map from 'ol/Map'
import View from 'ol/View'
import { fromLonLat } from 'ol/proj'
import ScaleLine from 'ol/control/ScaleLine'
import Zoom from 'ol/control/Zoom'

const mapRef = ref(null)
// 使用普通变量，不要用 ref() 包裹 Map 实例
let map = null

onMounted(() => {
  map = new Map({
    target: mapRef.value,
    layers: [
      createTianDiTuLayer('vec_w', TK),
      createTianDiTuLayer('cva_w', TK)
    ],
    view: new View({
      center: fromLonLat([117.0, 36.65]),
      zoom: 10,
      minZoom: 1,
      maxZoom: 18,
    }),
    controls: [new ScaleLine(), new Zoom()]
  })
})

onBeforeUnmount(() => {
  // 销毁地图：解除 DOM 绑定，释放内存
  map?.setTarget(null)
  map = null
})
</script>

<style scoped>
.map-container {
  width: 100%;
  height: 100%;
}
</style>

封装 useMap Composable ​

将地图初始化逻辑抽为可复用的组合式函数，多个页面可以共享同一套初始化逻辑：

js
// composables/useMap.js
import { ref, onMounted, onBeforeUnmount } from 'vue'
import Map from 'ol/Map'
import View from 'ol/View'
import { fromLonLat } from 'ol/proj'
import ScaleLine from 'ol/control/ScaleLine'
import Zoom from 'ol/control/Zoom'

export function useMap(options = {}) {
  const mapRef = ref(null)
  let map = null

  const {
    center = [117.0, 36.65],
    zoom = 10,
    layers = [],
  } = options

  onMounted(() => {
    map = new Map({
      target: mapRef.value,
      layers,
      view: new View({
        center: fromLonLat(center),
        zoom,
        minZoom: 1,
        maxZoom: 18,
      }),
      controls: [new ScaleLine(), new Zoom()]
    })
  })

  onBeforeUnmount(() => {
    map?.setTarget(null)
    map = null
  })

  // 返回 getter 函数而非直接返回 map，因为 map 在 onMounted 后才有值
  const getMap = () => map

  return { mapRef, getMap }
}

在组件中使用：

vue
<template>
  <div ref="mapRef" class="map-container" />
</template>

<script setup>
import { useMap } from '@/composables/useMap'

const { mapRef, getMap } = useMap({
  center: [117.0, 36.65],
  zoom: 12,
  layers: [vecLayer, cvaLayer]
})
</script>

点位标记 ​

Feature 与 Style 的关系 ​

在 OpenLayers 中，一个点位由三层结构组成：

  1. Feature（要素）：逻辑对象，携带几何信息和自定义业务属性
  2. Geometry（几何体）：定义空间位置，点位使用 Point
  3. Style（样式）：决定渲染外观，点位通常使用 Icon

样式可以设置在 Feature 上（feature.setStyle()），也可以设置在 VectorLayer 上。设置在图层上时支持传入函数，根据每个 Feature 的属性动态返回不同样式，适合批量点位场景。

基础点位 ​

js
function addMarker(map, { longitude, latitude, iconSrc, name }) {
  const feature = new Feature({
    geometry: new Point(fromLonLat([longitude, latitude])),
    name // 自定义属性，可通过 feature.get('name') 读取
  })

  const layer = new VectorLayer({
    source: new VectorSource({ features: [feature] }),
    zIndex: 100,
    style: new Style({
      image: new Icon({
        src: iconSrc,
        scale: 1,
        anchor: [0.5, 0.5],       // 图标锚点：[0.5, 0.5] 表示图标中心对准坐标点
        anchorXUnits: 'fraction',  // 锚点单位：fraction 为比例（0~1），pixels 为像素
        anchorYUnits: 'fraction'
      })
    })
  })

  map.addLayer(layer)
  return { layer, feature }
}

anchor 说明：[0.5, 1] 表示图标底部中心对准坐标点，常用于"大头针"类图标；[0.5, 0.5] 表示图标正中心对准坐标点，适合圆形标记。

动态样式点位 ​

当不同点位需要显示不同图标时，将图层的 style 设为函数。OpenLayers 在渲染每个 Feature 时会调用该函数，传入当前 Feature 作为参数：

js
const layer = new VectorLayer({
  source: new VectorSource({ features }),
  style: (feature) => {
    const status = feature.get('status')
    return new Style({
      image: new Icon({
        src: status === 'online' ? onlineIcon : offlineIcon,
        scale: 1,
      })
    })
  }
})

Vue3 点位标记组件 ​

vue
<template>
  <div ref="mapRef" class="map-container" />
</template>

<script setup>
import { ref, watch, onMounted, onBeforeUnmount } from 'vue'
import { useMap } from '@/composables/useMap'
import Feature from 'ol/Feature'
import Point from 'ol/geom/Point'
import { fromLonLat } from 'ol/proj'
import VectorSource from 'ol/source/Vector'
import VectorLayer from 'ol/layer/Vector'
import Style from 'ol/style/Style'
import Icon from 'ol/style/Icon'

const props = defineProps({
  points: { type: Array, default: () => [] }
})

const { mapRef, getMap } = useMap({ center: [117.0, 36.65], zoom: 10 })
let markerLayer = null

// 监听点位数据变化，自动更新地图
watch(() => props.points, (newPoints) => {
  updateMarkers(newPoints)
}, { deep: true })

function updateMarkers(points) {
  const map = getMap()
  if (!map) return

  // 清除旧图层
  if (markerLayer) {
    map.removeLayer(markerLayer)
  }

  const features = points.map(p => new Feature({
    geometry: new Point(fromLonLat([p.longitude, p.latitude])),
    name: p.name,
    status: p.status
  }))

  markerLayer = new VectorLayer({
    source: new VectorSource({ features }),
    style: (feature) => new Style({
      image: new Icon({
        src: feature.get('status') === 'online' ? onlineIcon : offlineIcon,
        scale: 1,
        anchor: [0.5, 0.5],
        anchorXUnits: 'fraction',
        anchorYUnits: 'fraction'
      })
    })
  })

  map.addLayer(markerLayer)
}

onMounted(() => {
  // 地图就绪后渲染初始点位
  if (props.points.length) {
    updateMarkers(props.points)
  }
})
</script>

Overlay 弹窗 ​

Overlay 与 VectorLayer 的本质区别在于：图层通过 Canvas 绘制图形，而 Overlay 是将真实的 DOM 元素锚定到地图坐标上。因此 Overlay 适合展示富文本内容（HTML 弹窗、表单、视频播放器等），可以直接使用 Vue 组件作为弹窗内容。

Overlay 关键配置 ​

属性说明
element要锚定的 DOM 元素
positioning元素相对于坐标点的定位方式，如 bottom-center 表示元素底部中心对齐坐标
offset[x, y] 像素偏移，用于微调位置
stopEvent是否阻止鼠标事件穿透到地图，弹窗场景通常设为 true
autoPan弹窗超出视口时是否自动平移地图使其可见

Vue3 弹窗组件 ​

vue
<template>
  <div ref="mapRef" class="map-container">
    <!-- 弹窗DOM，用 v-show 控制显隐 -->
    <div ref="popupRef" v-show="popupVisible" class="map-popup">
      <h4>{{ popupData.name }}</h4>
      <p>状态：{{ popupData.status }}</p>
      <button @click="closePopup">关闭</button>
    </div>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import Overlay from 'ol/Overlay'

const popupRef = ref(null)
const popupVisible = ref(false)
const popupData = ref({ name: '', status: '' })

let popupOverlay = null

onMounted(() => {
  const map = getMap()

  popupOverlay = new Overlay({
    element: popupRef.value,
    positioning: 'bottom-center',
    offset: [0, -20],
    stopEvent: true,
    autoPan: true
  })
  map.addOverlay(popupOverlay)

  // 点击地图要素时显示弹窗
  map.on('click', (event) => {
    const feature = map.forEachFeatureAtPixel(event.pixel, f => f)
    if (feature) {
      popupData.value = {
        name: feature.get('name'),
        status: feature.get('status')
      }
      popupOverlay.setPosition(feature.getGeometry().getCoordinates())
      popupVisible.value = true
    } else {
      closePopup()
    }
  })
})

function closePopup() {
  popupOverlay?.setPosition(undefined) // 传入 undefined 隐藏 Overlay
  popupVisible.value = false
}
</script>

setPosition(undefined) 是 Overlay 的标准隐藏方式，比操作 DOM display 更可靠，因为它会同时从地图的渲染循环中移除该 Overlay。

点位聚合（Cluster） ​

当点位数量达到成百上千时，逐个渲染会导致性能下降且视觉混乱。Cluster 数据源会自动将空间上接近的点位合并为一个聚合点，缩放放大后再逐步散开。

Cluster 包裹在普通 VectorSource 外层，其 distance 属性控制聚合半径（单位为像素）。聚合后的 Feature 通过 feature.get('features') 可获取到它包含的所有原始 Feature 数组。

vue
<script setup>
import Cluster from 'ol/source/Cluster'

function createClusterLayer(features, map) {
  const clusterSource = new Cluster({
    distance: 60, // 60 像素范围内的点位会被聚合
    source: new VectorSource({ features })
  })

  const layer = new VectorLayer({
    source: clusterSource,
    style: (feature) => {
      // feature.get('features') 返回被聚合的原始 Feature 数组
      const count = feature.get('features').length

      if (count === 1) {
        // 只有一个点，显示原始图标
        return new Style({ image: new Icon({ src: markerIcon }) })
      }
      // 多个点，显示聚合数量
      return new Style({
        image: new Icon({ src: clusterIcon }),
        text: new Text({
          text: count.toString(),
          fill: new Fill({ color: '#FFF' }),
          backgroundFill: new Fill({ color: '#0052D9' }),
          padding: [2, 4, 2, 4]
        })
      })
    }
  })

  map.addLayer(layer)
  return layer
}
</script>

动态聚合距离 ​

随着地图缩放，可以动态调整聚合行为——放大到一定级别后关闭聚合，让每个点位独立显示：

js
// View 的 change:resolution 事件在每次缩放时触发
map.getView().on('change:resolution', () => {
  const zoom = map.getView().getZoom()
  const source = clusterLayer.getSource()
  // 缩放到 16 级及以上时关闭聚合
  source.setDistance(zoom >= 16 ? 0 : 60)
})

图形绘制 ​

OpenLayers 的 Draw 交互是绘制功能的核心。它接管鼠标操作，让用户在地图上通过点击/拖拽绘制几何图形。关键参数说明：

参数说明
source绘制完成的 Feature 自动添加到此数据源
type绘制类型：Point、LineString、Polygon、Circle
geometryFunction自定义几何体生成函数，如 createBox() 将 Circle 变为矩形
style绘制过程中的临时样式

Draw 提供两个关键事件：drawstart（开始绘制时触发）和 drawend（绘制完成时触发，通过 event.feature 获取绘制的图形）。

封装 useDraw Composable ​

js
// composables/useDraw.js
import { onBeforeUnmount } from 'vue'
import Draw, { createBox } from 'ol/interaction/Draw'
import VectorSource from 'ol/source/Vector'
import VectorLayer from 'ol/layer/Vector'
import { toLonLat } from 'ol/proj'
import Style from 'ol/style/Style'
import Fill from 'ol/style/Fill'
import Stroke from 'ol/style/Stroke'

const drawStyle = new Style({
  fill: new Fill({ color: 'rgba(22, 99, 227, 0.1)' }),
  stroke: new Stroke({ color: '#1663E3', width: 2, lineDash: [8, 5] })
})

export function useDraw(getMap) {
  let currentDraw = null
  let currentLayer = null

  /** 创建绘制交互 */
  function startDraw(type, callback, options = {}) {
    stopDraw()
    const map = getMap()
    if (!map) return

    const source = new VectorSource()
    currentLayer = new VectorLayer({ source, style: drawStyle, zIndex: 200 })
    map.addLayer(currentLayer)

    const drawOptions = { source, type, style: drawStyle }

    // 矩形特殊处理：type 设为 Circle，通过 geometryFunction 转为矩形
    if (type === 'Rectangle') {
      drawOptions.type = 'Circle'
      drawOptions.geometryFunction = createBox()
    }

    currentDraw = new Draw(drawOptions)
    currentDraw.on('drawend', event => {
      const geometry = event.feature.getGeometry()
      callback(geometry)
      // 绘制完成后自动停止（单次绘制模式）
      stopDraw()
    })
    map.addInteraction(currentDraw)
  }

  /** 停止绘制并清理 */
  function stopDraw() {
    const map = getMap()
    if (!map) return
    if (currentDraw) {
      map.removeInteraction(currentDraw)
      currentDraw = null
    }
    if (currentLayer) {
      map.removeLayer(currentLayer)
      currentLayer = null
    }
  }

  // 组件卸载时自动清理
  onBeforeUnmount(() => stopDraw())

  return { startDraw, stopDraw }
}

在组件中使用 ​

vue
<template>
  <div ref="mapRef" class="map-container" />
  <div class="toolbar">
    <button @click="drawRect">矩形圈选</button>
    <button @click="drawCircle">圆形圈选</button>
    <button @click="drawPolygon">多边形圈选</button>
    <button @click="drawLine">折线绘制</button>
    <button @click="cancelDraw">取消</button>
  </div>
</template>

<script setup>
import { useMap } from '@/composables/useMap'
import { useDraw } from '@/composables/useDraw'
import { toLonLat } from 'ol/proj'

const { mapRef, getMap } = useMap()
const { startDraw, stopDraw } = useDraw(getMap)

function drawRect() {
  startDraw('Rectangle', (geometry) => {
    const coords = geometry.getCoordinates()[0]
    const lngs = coords.map(c => c[0])
    const lats = coords.map(c => c[1])
    const topLeft = toLonLat([Math.min(...lngs), Math.max(...lats)])
    const bottomRight = toLonLat([Math.max(...lngs), Math.min(...lats)])
    console.log('矩形范围:', { topLeft, bottomRight })
  })
}

function drawCircle() {
  startDraw('Circle', (geometry) => {
    const center = toLonLat(geometry.getCenter())
    const radius = geometry.getRadius()
    console.log('圆心:', center, '半径:', radius, '米')
  })
}

function drawPolygon() {
  startDraw('Polygon', (geometry) => {
    // 去除闭合点（Polygon 自动闭合，最后一个点与第一个重复）
    const points = geometry.getCoordinates()[0].slice(0, -1).map(c => toLonLat(c))
    console.log('多边形顶点:', points)
  })
}

function drawLine() {
  startDraw('LineString', (geometry) => {
    const points = geometry.getCoordinates().map(c => toLonLat(c))
    console.log('折线节点:', points)
  })
}

function cancelDraw() {
  stopDraw()
}
</script>

矩形绘制原理：OpenLayers 没有内置 Rectangle 类型。createBox() 是官方提供的 geometryFunction，它将 Circle 的两点操作（圆心+边界）映射为矩形的两点操作（对角线端点），最终生成 Polygon 几何体。

折线连线 ​

在巡检路线等场景中，将有序的坐标点用折线连接。LineString 接受坐标数组，按顺序连接各点：

js
function addLine(map, pointList) {
  const coords = pointList.map(p =>
    fromLonLat([Number(p.longitude), Number(p.latitude)])
  )
  if (coords.length < 2) return null

  const feature = new Feature({ geometry: new LineString(coords) })
  const layer = new VectorLayer({
    source: new VectorSource({ features: [feature] }),
    style: new Style({
      stroke: new Stroke({ color: '#1663E3', width: 3 })
    })
  })
  map.addLayer(layer)
  return layer
}

Stroke 的 lineDash 属性可以设置虚线样式，如 [10, 5] 表示 10px 实线 + 5px 间隔。

扇形展示 ​

用于显示摄像头可视范围，核心思路：以摄像头位置为顶点，根据可视距离、朝向和水平视场角，在弧线上每隔几度取一个点，这些点与顶点连接构成一个近似扇形的 Polygon。

坐标计算原理 ​

扇形的弧线部分是地球表面的一段弧，需要使用 大圆弧公式（Haversine） 根据起点、距离和方位角计算弧上每个点的经纬度坐标。方位角 0° 为正北，顺时针增长。

js
/**
 * 大圆弧计算：根据起点、距离、方位角求目标点
 * 计算结果直接转为墨卡托坐标，方便传入 OpenLayers
 */
function calcPointByBearing(startLonLat, distance, bearing) {
  const [lon, lat] = startLonLat
  const R = 6371000 // 地球半径（米）
  const rad = Math.PI / 180
  const bearingRad = bearing * rad
  const lat1 = lat * rad
  const lon1 = lon * rad

  const lat2 = Math.asin(
    Math.sin(lat1) * Math.cos(distance / R) +
    Math.cos(lat1) * Math.sin(distance / R) * Math.cos(bearingRad)
  )
  const lon2 = lon1 + Math.atan2(
    Math.sin(bearingRad) * Math.sin(distance / R) * Math.cos(lat1),
    Math.cos(distance / R) - Math.sin(lat1) * Math.sin(lat2)
  )

  return fromLonLat([lon2 / rad, lat2 / rad])
}

/**
 * 生成扇形多边形坐标
 * @param vertex       摄像头经纬度 [lon, lat]
 * @param range        可视距离（米）
 * @param direction    朝向角度（0=正北，顺时针）
 * @param fieldAngle   水平视场角（度）
 */
function calcSectorCoords(vertex, range, direction, fieldAngle) {
  const center = fromLonLat(vertex)
  const start = direction - fieldAngle / 2
  const end = direction + fieldAngle / 2
  const step = 5 // 弧线精度：每 5° 取一点，值越小越平滑

  const points = [center]
  for (let angle = start; angle <= end; angle += step) {
    points.push(calcPointByBearing(vertex, range, angle))
  }
  // 确保终止角度被包含
  if ((end - start) % step !== 0) {
    points.push(calcPointByBearing(vertex, range, end))
  }
  points.push(center) // 闭合
  return points
}

绘制扇形 ​

纯色填充：使用标准的 Fill + Stroke 样式，简单直接。

js
function drawSector(map, { vertex, range, direction, fieldAngle }) {
  const coords = calcSectorCoords(vertex, range, direction, fieldAngle)
  const feature = new Feature({ geometry: new Polygon([coords]) })
  feature.setStyle(new Style({
    fill: new Fill({ color: 'rgba(255, 0, 0, 0.2)' }),
    stroke: new Stroke({ color: 'rgba(255, 0, 0, 0.5)', width: 2 })
  }))

  const layer = new VectorLayer({
    source: new VectorSource({ features: [feature] })
  })
  map.addLayer(layer)
  return layer
}

径向渐变：利用 Style 的 renderer 回调直接操作 Canvas 2D Context，实现从中心到边缘的颜色渐淡效果，视觉上更接近真实的可视范围衰减。

js
feature.setStyle(new Style({
  renderer: (pixelCoords, state) => {
    const ring = pixelCoords?.[0]
    if (!ring || ring.length < 3) return

    const [cx, cy] = ring[0] // 顶点在 Canvas 上的像素位置
    let maxR = 0
    for (let i = 1; i < ring.length; i++) {
      maxR = Math.max(maxR, Math.hypot(ring[i][0] - cx, ring[i][1] - cy))
    }

    const ctx = state.context
    const gradient = ctx.createRadialGradient(cx, cy, 0, cx, cy, maxR)
    gradient.addColorStop(0, 'rgba(255, 0, 0, 0.35)')
    gradient.addColorStop(1, 'rgba(255, 0, 0, 0.05)')

    ctx.save()
    ctx.beginPath()
    ring.forEach(([x, y], i) => i === 0 ? ctx.moveTo(x, y) : ctx.lineTo(x, y))
    ctx.closePath()
    ctx.fillStyle = gradient
    ctx.fill()
    ctx.strokeStyle = 'rgba(255, 0, 0, 0.5)'
    ctx.stroke()
    ctx.restore()
  }
}))

renderer 回调说明：这是 OpenLayers 的高级特性，绕过了标准样式系统，直接拿到 Canvas Context 进行自由绘制。pixelCoords 是几何体各顶点转换后的屏幕像素坐标，state.context 是当前 Canvas 的 2D 绑定上下文。每次地图重绘（平移、缩放）都会重新调用此函数。

Vue3 扇形组件 ​

<script setup>
import { watch } from 'vue'

const props = defineProps({

地图交互 ​

要素检测 —— forEachFeatureAtPixel ​

这是 OpenLayers 中最常用的交互入口。给定一个屏幕像素坐标，遍历该位置下的所有要素，返回第一个匹配的结果：

js
map.forEachFeatureAtPixel(pixel, callback, options)
参数说明
pixel屏幕像素坐标 [x, y]，通常来自事件对象的 event.pixel
callback命中要素时的回调，返回值作为整体返回值（返回非 undefined 则停止遍历）
options.hitTolerance容差像素数，扩大点击的检测范围
options.layerFilter图层过滤函数，限定只检测哪些图层

Hover 提示 ​

在 pointermove 事件中检测当前鼠标下方是否有要素，有则显示 Tooltip，无则隐藏：

vue
<script setup>
import { ref, onMounted } from 'vue'
import Overlay from 'ol/Overlay'

const tooltipRef = ref(null)
const tooltipText = ref('')
let tooltipOverlay = null

onMounted(() => {
  const map = getMap()

  tooltipOverlay = new Overlay({
    element: tooltipRef.value,
    positioning: 'bottom-center',
    offset: [0, -15]
  })
  map.addOverlay(tooltipOverlay)

  map.on('pointermove', (event) => {
    const feature = map.forEachFeatureAtPixel(event.pixel, f => f, {
      hitTolerance: 10,
      layerFilter: layer => layer === markerLayer
    })

    if (feature) {
      tooltipText.value = feature.get('name')
      tooltipOverlay.setPosition(feature.getGeometry().getCoordinates())
    } else {
      tooltipOverlay.setPosition(undefined)
    }
  })
})
</script>

拖拽点位（Translate） ​

Translate 交互允许用户拖拽地图上的要素。它提供三个事件：

  • translatestart：拖拽开始
  • translating：拖拽过程中持续触发（可用于实时更新关联图形，如扇形跟随点位移动）
  • translateend：拖拽结束，获取最终坐标
vue
<script setup>
import Translate from 'ol/interaction/Translate'
import Collection from 'ol/Collection'
import { toLonLat } from 'ol/proj'

const emit = defineEmits(['drag-end'])
let translate = null

function enableDrag(features) {
  const map = getMap()
  if (translate) map.removeInteraction(translate)

  translate = new Translate({
    features: new Collection(features) // 指定可拖拽的要素集合
  })

  translate.on('translating', (event) => {
    // 拖拽过程中实时更新扇形位置
    const coord = event.features.item(0).getGeometry().getCoordinates()
    updateSectorPosition(coord)
  })

  translate.on('translateend', (event) => {
    const coord = event.features.item(0).getGeometry().getCoordinates()
    const [lon, lat] = toLonLat(coord)
    emit('drag-end', { longitude: lon, latitude: lat })
  })

  map.addInteraction(translate)
}
</script>

测距工具 ​

OpenLayers 的 ol/sphere 模块提供 getLength 函数，可计算线段在地球表面的真实距离（考虑了地球曲率），结果单位为米。测距工具的实现思路是：使用 Draw 交互绘制折线，每新增一个节点就计算并标注累计距离。

vue
<template>
  <button @click="startMeasure">测距</button>
  <button @click="stopMeasure">停止</button>
</template>

<script setup>
import Draw from 'ol/interaction/Draw'
import Overlay from 'ol/Overlay'
import LineString from 'ol/geom/LineString'
import { getLength } from 'ol/sphere'
import { unByKey } from 'ol/Observable'

let draw = null
let measureLayer = null
const overlays = []

function formatDist(m) {
  return m > 1000 ? (m / 1000).toFixed(2) + ' km' : Math.round(m) + ' m'
}

function addLabel(coord, text) {
  const el = document.createElement('div')
  el.className = 'measure-label'
  el.textContent = text

  const overlay = new Overlay({
    element: el,
    positioning: 'bottom-center',
    offset: [0, -10]
  })
  overlay.setPosition(coord)
  getMap().addOverlay(overlay)
  overlays.push(overlay)
}

function startMeasure() {
  stopMeasure()
  const map = getMap()
  const source = new VectorSource()
  measureLayer = new VectorLayer({ source, zIndex: 200 })
  map.addLayer(measureLayer)

  draw = new Draw({ source, type: 'LineString' })
  let coords = []
  let listenerKey = null

  draw.on('drawstart', (evt) => {
    coords = [evt.feature.getGeometry().getFirstCoordinate()]
    addLabel(coords[0], '起点')

    // 监听几何体实时变化
    listenerKey = evt.feature.getGeometry().on('change', (e) => {
      const current = e.target.getCoordinates()
      if (current.length > coords.length) {
        const newCoord = current[current.length - 2]
        coords.push(newCoord)
        const dist = getLength(new LineString(coords), { projection: 'EPSG:3857' })
        addLabel(newCoord, formatDist(dist))
      }
    })
  })

  draw.on('drawend', (evt) => {
    if (listenerKey) unByKey(listenerKey) // 解除几何体监听
    const total = getLength(evt.feature.getGeometry(), { projection: 'EPSG:3857' })
    addLabel(evt.feature.getGeometry().getLastCoordinate(), '总长：' + formatDist(total))
    map.removeInteraction(draw)
  })

  map.addInteraction(draw)
}

function stopMeasure() {
  const map = getMap()
  if (!map) return
  if (draw) { map.removeInteraction(draw); draw = null }
  if (measureLayer) { map.removeLayer(measureLayer); measureLayer = null }
  overlays.forEach(o => map.removeOverlay(o))
  overlays.length = 0
}
</script>

unByKey 的作用：OpenLayers 的事件监听返回一个 key，通过 unByKey(key) 可以精确移除该监听。这在绘制完成后解除 geometry.on('change') 监听时尤为重要，避免内存泄漏。

  duration: 500
})

坐标转换 ​

贯穿 OpenLayers 使用的最核心函数：

函数方向使用场景
fromLonLat([lon, lat])WGS84 → 墨卡托将业务数据（经纬度）传入地图
toLonLat([x, y])墨卡托 → WGS84从地图中取出坐标提交给后端
js
import { fromLonLat, toLonLat } from 'ol/proj'

// 创建点位时：经纬度 → 墨卡托
new Point(fromLonLat([117.0, 36.65]))

// 绘制完成后取坐标：墨卡托 → 经纬度
const [lon, lat] = toLonLat(feature.getGeometry().getCoordinates())

简记：进(from) 出(to)——数据进入地图用 fromLonLat，数据离开地图用 toLonLat。
